LEGACY SAFARIS LTD – COMPLETE MULTI-PAGE WEBSITE
================================================

STRUCTURE:
- 7 HTML pages (home, about, tours, gallery, contact, destinations, blog)
- Shared header & footer via partials/ (loaded with JavaScript includes)
- Separate CSS (css/style.css) and JS (js/include.js + js/main.js)

HOW TO RUN:
1. Place all files exactly as shown in the folder tree.
2. Serve the folder using a local web server:
   - Using VS Code Live Server extension: right-click index.html → Open with Live Server
   - OR run `npx http-server` in the terminal
   - OR use Python: `python -m http.server 8000`
3. Open http://localhost:8000

FEATURES:
- Responsive design (mobile-friendly)
- Lightbox gallery
- Contact form with validation
- Tour cards with "Book Enquiry" that pre-fills the contact page
- Dynamic active navigation highlighting
- Smooth hover effects and safari-themed color palette
